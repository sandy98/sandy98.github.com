Node Simple Router a.k.a. NSR

Yet another minimalistic router for node.js

When all you need is routing...

The lightweight router for the lightweight web app.

Even though, should you need CGI or SCGI invoking, proxy passing, file uploading, or even simple templating, you got them out of the box.

No dependencies, all you need is NodeJS itself, NSR and off you go!

And we point to keep it that way, simple and independent, nevertheless more functional in each new incarnation.
